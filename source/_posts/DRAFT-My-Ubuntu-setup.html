I format my laptop quite often (almost in line with the Ubuntu six monthly release cycle) which keeps my system nice and clean and also gives me good confidence in my backups. The majority of the stuff I need on my laptop is text (code, dot files, vim config, etc.) and so they live on GitHub, music is sync'ed to my phone and I try to keep photos off the laptop and make sure I put them onto my PC's RAID.

This post provides a basic outline of what I do to re-install Ubuntu on my laptop.

Install the latest Ubuntu
-----------------------

- Boot off the USB
- Install Ubuntu over the existing Linux partition, but leave the old windows recovery partition... just in case
- Enable the third party package repos and do a software update
- Configure power settings, two finger scroll, samba sharing, alt+ctrl+backspace (restart X), fstab, printer

Install Gnome-Shell
-------------------

Gnome Shell is my desktop environment of choice, so that's the first thing I install. While I am a big fan of most parts, there are a few things that I don't like, but I fix that with extensions.

```
sudo apt-get install gnome-shell
```

- Restart X with Gnome
- Goto extensions.gnome.org to install these plugins:
  - https://extensions.gnome.org/extension/67/dash-click-fix/
  - https://extensions.gnome.org/extension/112/remove-accesibility/
  - https://extensions.gnome.org/extension/5/alternative-status-menu/
  - https://extensions.gnome.org/extension/15/alternatetab/
https://extensions.gnome.org/extension/517/caffeine/
https://extensions.gnome.org/extension/235/desktop-scroller-left-and-overview-version/
https://extensions.gnome.org/extension/310/alt-tab-workspace/


Install applications
------------------

```
sudo apt-get install terminator git build-essential vim-gnome exuberant-ctags tmux screen curl openssh-server vlc gimp skype calibre tree meld ack-grep wireshark netcat nmap gparted
```

I also install these applications from the official sites:
- Google Chrome (instead of Chromium which is available in apt)
- Dropbox


Pull down my configs
----------------------

 dot files, vim config


Setup dev environment
------------------------

- RVM or RBenv
- Ruby
- Gems (Rails, Bundler, etc.)
- MySQL or Postgres
